## 🔧 How to Use

### Run Stress Test
```bash
cd /root/thaiVaiEcom2.0
# Development stack with source mounts
docker compose -f docker-compose.dev.yml up -d
# Execute stress test
docker exec thaivaiecom20-medusa-1 npx medusa exec ./src/scripts/stress-test-products.ts
```

### View Results
```bash
# Database verification
docker exec thaivaiecom20-postgres-1 psql -U postgres -d medusa-store -t -c \
  "SELECT COUNT(*) FROM product WHERE title LIKE 'Stress Test%';"
# Shows: 500 ✅
```

### Clean Up Test Data
```bash
# Option 1: Re-run seed
docker exec thaivaiecom20-medusa-1 npx medusa exec ./src/scripts/seed.ts

# Option 2: Delete test products
docker exec thaivaiecom20-postgres-1 psql -U postgres -d medusa-store -c \
  "DELETE FROM product WHERE title LIKE 'Stress Test%';"
```
