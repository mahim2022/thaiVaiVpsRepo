# Custom Orders - User Guide
**For Customers and Admin Staff**

---

## What is Custom Orders?

Custom Orders allows you to request custom-made products with specific requirements. Instead of buying from our catalog, you can describe exactly what you need and our team will review your request and provide a quote or update.

---

## For Customers: How to Make a Custom Order Request

### Step 1: Go to Custom Orders Page
1. Log into your account
2. Go to **Account** → **Custom Orders**
3. Click **"New Request"**

### Step 2: Fill Out Your Request
You need to provide:

**Title** (Required)
- A short name for your request
- Example: "Custom T-Shirt with Logo"
- Maximum 200 characters

**Description** (Required)
- Detailed explanation of what you need
- Include materials, colors, sizes, quantities
- Example: "I need 50 red cotton t-shirts with embroidered company logo on the front. Size range from XS to XXL."

**Attach Images** (Optional but helpful)
- Upload photos showing what you want
- You can add up to 5 images
- Each image must be less than 5MB
- Supported formats: PNG, JPG, JPEG, WEBP, GIF
- Example: Upload a photo of your logo or reference design

### Step 3: Submit Your Request
1. Click **"Submit request"**
2. You'll see a success message
3. Your request is now submitted to our admin team

### Step 4: Wait for Admin Response
Your request will go through these stages:

| Status | What It Means |
|--------|---------------|
| 🔵 **Submitted** | We received your request. Team is reviewing. |
| 🟡 **In Review** | We're actively looking at your request. |
| 🟢 **Replied** | We have feedback, quote, or an update for you. |
| ⚫ **Closed** | Order complete or request closed. |

### Step 5: Check for Admin Reply
1. Go back to **Account** → **Custom Orders**
2. Click on your request
3. Scroll down to see **"Admin reply"** box when available
4. This will contain:
   - Pricing information
   - Timeline/delivery date
   - Questions we need answered
   - Next steps

## For Admin Staff: How to Reply to Custom Orders

### Access the Admin Dashboard

1. Log in as admin
2. Go to **Admin** → **Custom Orders**
3. You'll see a list of all customer requests

### Review a Request

1. Click on any customer request
2. You'll see:
   - Customer's title and description
   - Any attachments (photos) they uploaded
   - Order status

### Reply to a Request

There are 3 ways to respond:

#### Option 1: Initial Review Assessment
**When:** Customer's request is "Submitted" and you're just getting started

1. Set Status to: **In Review**
2. Optional: Add a message like "Thanks for your request. We are reviewing it now."
3. Click **Save**
4. Customer will see your status update

#### Option 2: Provide Information/Quote
**When:** You have pricing, timeline, or need more details

1. Set Status to: **Replied**
2. Add your message in the reply box
3. Example replies:
   - "We can make 50 units at $25 each. Timeline: 2 weeks."
   - "We need clarification on the logo size. Should it be 2 inches or 4 inches?"
   - "We have a question: do you prefer polyester or cotton blend?"
4. Click **Save**
5. Customer receives notification and sees your reply immediately

#### Option 3: Go Back to Review
**When:** Customer replies with more info and you need to re-assess

1. Set Status back to: **In Review**
2. Customer knows you're actively reviewing again
3. Don't add a message unless you want to update them

#### Option 4: Complete the Order
**When:** Everything is decided and order is processed

1. Set Status to: **Closed**
2. Do NOT add a new reply when closing
3. Click **Save**
4. Customer will see the order is complete

## Tips for Customers

✅ **DO:**
- Be as detailed as possible in your description
- Upload clear, high-quality photos
- Include all measurements and quantities
- Explain the exact colors and materials you want
- Mention any special requirements (rush order, budget limits, etc.)

❌ **DON'T:**
- Upload very large files (keep images under 5MB)
- Upload non-image files (PDFs, documents won't work)
- Upload more than 5 images per request
- Leave the description blank

**Best Practice:** The more details you provide upfront, the faster the admin team can respond with accurate information.

---

## Tips for Admin Staff

✅ **DO:**
- Respond promptly to new requests (within 24 hours if possible)
- Ask clarifying questions if needed (use "In Review" status)
- Provide clear pricing and timelines in your reply
- Keep messages professional and friendly
- Explain any limitations or constraints

❌ **DON'T:**
- Leave requests in "Submitted" status for too long
- Close an order without confirming all details with customer
- Try to add a reply when closing (system won't allow it)
- Send incomplete information

**Best Practice:** Use "In Review" for initial assessment, "Replied" when you have concrete information, and only "Closed" when the order is finalized.

## Frequently Asked Questions

### Customer Questions

**Q: How long does it take to get a response?**
A: Our team typically responds within 24-48 hours. You'll see the status change to "In Review" when we're actively working on your request.

**Q: Can I attach videos or PDF files?**
A: No, only image files (PNG, JPG, JPEG, WEBP, GIF). If you need to share other documents, describe the requirements in the description field instead.

**Q: What if I need to provide more information?**
A: If our admin team needs clarification, they'll reply with "Replied" status and ask in their message. You can then submit a new custom order or reply directly.

**Q: How do I know when admin has replied?**
A: Check your account page - you'll see the status change from "Submitted" or "In Review" to "Replied". You should also receive an email notification.

**Q: Can I cancel my request?**
A: Contact customer support directly. Don't submit duplicate requests.

### Admin Questions

**Q: What should I do if the customer's request is unclear?**
A: Set status to "In Review" and add a message asking for clarification. Example: "Could you clarify the size of the embroidered logo? 2 inches or 4 inches?"

**Q: Can I edit a reply after sending it?**
A: No. If you make a mistake, set status back to "In Review", then create a new "Replied" message with the correct information.

**Q: What does "Closed" mean?**
A: The request is complete - either the order has been confirmed and placed, or we've rejected the request. It's a final status.

**Q: Can a closed order be reopened?**
A: No. If a customer wants to continue, they should submit a new custom order request.
